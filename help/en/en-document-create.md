# Creating a Document (Offer)

Create a personalized PDF offer file to present to your patients. This file can include the treatment plan, prices, before/after visuals, and your clinic introduction.

**Path:** `Cases > [Patient Name] > Document Pages`

## Page Selection

On the first screen, select which pages you want to include in the PDF.

* **Cover / Back:** Standard cover designs.
* **Intro:** Clinic and doctor introduction.
* **Treatment:** Planned procedures and price list.
* **Case Photos:** Case photos.
* **Diagnosis:** Tooth chart and diagnosis details.
